## Environments
```env
R7PLATFORM_WORKER_GRPC_SERVER=localhost:50051

R7PLATFORM_WORKER_REDIS_HOST=localhost
R7PLATFORM_WORKER_REDIS_PASSWORD=admin

R7PLATFORM_WORKER_LOG_REDIS_HOST=localhost
R7PLATFORM_WORKER_LOG_REDIS_PASSWORD=admin

R7PLATFORM_WORKER_NOTIFY_REDIS_HOST=localhost
R7PLATFORM_WORKER_NOTIFY_REDIS_PASSWORD=admin

R7PLATFORM_WORKER_CONCURRENCY=4
R7PLATFORM_WORKER_MAX_QUEUE_LIMIT=10
R7PLATFORM_WORKER_ZONE=R7QUEUE
R7PLATFORM_WORKER_GRPC_API_KEY=x.x.x
```

`R7PLATFORM_WORKER_ZONE` ชื่อของ ZONE ที่ต้องการจะรับข้อมูล

## Run

```shell
NODE_ENV=development \
R7PLATFORM_WORKER_ZONE=DEMOQUEUE \
R7PLATFORM_WORKER_REDIS_HOST=localhost \
R7PLATFORM_WORKER_REDIS_PASSWORD=xxxxx \
R7PLATFORM_WORKER_LOG_REDIS_HOST=localhost \
R7PLATFORM_WORKER_LOG_REDIS_PASSWORD=xxxxxx \
R7PLATFORM_WORKER_NOTIFY_REDIS_HOST=localhost \
R7PLATFORM_WORKER_NOTIFY_REDIS_PASSWORD=xxxx \
R7PLATFORM_WORKER_GRPC_SERVER=localhost:50051 \
R7PLATFORM_WORKER_GRPC_API_KEY=xx.xx.xx \
R7PLATFORM_WORKER_MAX_QUEUE_LIMIT=10 \
R7PLATFORM_WORKER_CONCURRENCY=4 \
npx nodemon
```
